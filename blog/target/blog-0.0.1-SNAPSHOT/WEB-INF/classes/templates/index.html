清史小记

越学习越觉得自己无知，需要去学习更多的东西。 一直对历史上的故事饶有兴致，今日听人讲到一个观点：历史不能称之为科学，因为任何人都无法去对历史进行重复检验，无法穿越回过去。那我们学习历史的意义何在？历史究竟能为我们带来些什么？
